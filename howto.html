How to use the Permission import module:

Setup in REDCap
  1. Enable the module.
  2. For projects with this module enabled you should see the 'Import permissions' link.
    Here you can find a template file. Download it, fill it in and upload it.
    Currently no mail is sent for each imported line.
    User names are checked before being added. You can disable this with the 'Check that the usernames exist.' checkbox.
